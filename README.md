# cos-front-com

## Project setup

yarn install

## Compiles and hot-reloads for development

yarn run serve

### Compiles and minifies for production

yarn run build

### Run your tests

yarn run test

### Lints and fixes files

yarn run lint


### Git分支

1. master是开发分支, 代码对应研发环境的现状
2. qa是测试, 代码测试环境的现状
3. prod是生产分支, 代码对应生产环境的现状
